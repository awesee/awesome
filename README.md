## Awesome tools

## Install

```
go install github.com/awesee/awesome@latest
```

## Linux

```
curl -sSL https://github.com/awesee/awesome/releases/latest/download/awesome-linux-amd64 -o /usr/local/bin/awesome
```

## Help

```
Awesome tools

Usage:
  awesome [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  conc        Conc command

  redis       Lookup redis keys without expire
  update      Install the latest version of awesome

Flags:
  -h, --help      help for awesome
```
